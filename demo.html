TextColor.js

一个轻量级的JavaScript插件，可以通过这个插件对字母、单词或句子进行染色，无需依赖jQuery。


1、基本使用

HTML

<div id="demo1">
    <h1>TextColor</h1>
</div>

JavaScript

TextColor.listen({seletor:"#demo1"});

提供3个参数：
1.seletor：选择器
2.splitter：分割方式，3个参数，"char","word","line".默认为"char".
3.attr:是否添加属性。默认为false。


Result

TextColor

2、分割单词

HTML

<div id="demo2">
    <h1>This is TextColor</h1>
</div>

JavaScript

TextColor.listen({seletor:"#demo2",splitter:"word",attr:true});

Result

This is TextColor

3、划分句子(以<br>结束的句子)

HTML

<div id="demo3">
    <p>
         This is an amazing<br>
         Revolution in Typography.<br>
         The possibilities are endless:<br>
         Coloring, Vertical spacing, and Kerning.<br>
    </p>
</div>

JavaScript

TextColor.listen({seletor:'#demo3',splitter:"line"});

Result

This is an amazing
Revolution in Typography.
The possibilities are endless:
Coloring, Vertical spacing, and Kerning.

4、class类调用

HTML

<div class="demo4">
    <h1>This is TextColor</h1>
</div>
<div class="demo4">
    <h1>中文文字</h1>
</div>

JavaScript

TextColor.listen({seletor:".demo4"});

Result

This is TextColor

中文文字
