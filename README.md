## TextColor.js
> 一个轻量级的JavaScript插件，可以通过这个插件对字母、单词或句子进行染色，无需依赖jQuery。

- 调用方法

```javascript
TextColor.listen({seletor:"#demo1"});
```

- 提供3个参数：
  - 1.seletor：选择器
  - 2.splitter：分割方式，3个参数，"char","word","line".默认为"char".
  - 3.attr:是否添加属性。默认为false。
  
- 更多请查看demo.html
